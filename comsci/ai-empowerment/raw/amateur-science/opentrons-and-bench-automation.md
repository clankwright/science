# Opentrons OT-2 / Flex and LLM-Generated Lab Automation

## What it is

Opentrons makes the OT-2 (~$10K) and Flex (~$25K) liquid-handling robots whose protocols are written in Python via the open-source `opentrons` API. The amateur unlock is that as of 2024-2025, Opentrons ships LLM-based protocol generation: describe a protocol in English, get back runnable Python that the robot will execute. Combined with community biolabs (Genspace NYC, BioCurious Bay Area, BUGSS Baltimore) that own these robots, an amateur can run a real wet-lab experiment without buying their own equipment or having institutional access.

## Specific unlocks

- Describe a serial dilution, ELISA, plasmid prep, or PCR setup in natural language via Opentrons' AI tool; get back a complete `.py` protocol file, simulate it in their cloud simulator, then book an OT-2 at a community lab to run it.
- A protein engineer with no automation experience can set up a 96-well-plate combinatorial mutagenesis experiment by prompting an LLM with their goals, the available enzymes, and the deck layout (the Sept 2025 bioRxiv preprint "Autonomous liquid-handling robotics scripting through large language models" demonstrates this for protein engineering with safety guardrails).
- Decompose a JoVE protocol PDF or a literature methods section into discrete `pipette.transfer(...)` calls via an LLM, catching the common "the paper omits the centrifuge step" failure mode.
- Run protocols generated by Coscientist-class agents (CMU's GPT-4 powered planner) on OT-2 hardware; the loop "LLM plans -> Python generated -> robot runs -> data captured -> LLM analyzes" is now demonstrated end-to-end.
- Hop between Genspace (NYC), BioCurious (Sunnyvale), BUGSS (Baltimore), and roughly 60 other community labs listed on DIYbiosphere; each has shared automation hardware accessible to dues-paying members.

## Pre-AI baseline

Pre-2020, automating even a serial dilution required: (a) buy a Hamilton, Tecan, or Beckman robot ($50K-$300K), (b) hire a robotics technician or learn the vendor's bespoke scripting language, (c) maintain it. OT-2 in 2018 already democratized hardware to ~$10K and Python, but you still had to write the Python. The 2024 LLM tooling closed that last gap. Bench technique itself remains a skill, but protocol authoring is no longer a separate craft.

## Hardware / cost

- OT-2: ~$10K used / $12K new.
- Flex: ~$25-50K configured.
- Community lab membership: $50-200/month at Genspace, BioCurious; equipment access included.
- Opentrons cloud simulator: free.
- LLM protocol generation: free in their UI, or roll your own with Claude / GPT API + the `opentrons` Python docs.

## Maturity

- Opentrons hardware: production, used in pharma + academia.
- Opentrons LLM protocol generator: beta as of 2025; usable but requires human review of every protocol before running on real samples.
- Coscientist-style autonomous planners: research; impressive demos, narrow domains.
- Community-lab access model: production, has been working for >10 years.

## Where it breaks

- LLMs miss tip-rack reload events, deck-position collisions, and incompatible labware; simulator catches most but not all.
- Biological reagents have shelf-life and contamination considerations a code-only LLM does not track.
- The robot is good at moving liquid; it does not transform a fundamentally bad experimental design into a good one.
- Safety: the same Coscientist team that demonstrated Pd-catalyzed cross-coupling also demonstrated that prompting could elicit controlled-drug and weapons-of-concern syntheses. Community labs now publish use policies; OT-2 / Flex have no built-in chemistry filter.
