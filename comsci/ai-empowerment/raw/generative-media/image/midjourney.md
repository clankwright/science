## Vendor / access

Midjourney, Inc. Web app at midjourney.com (Discord interface deprecated as primary entry by mid-2025). Subscription only; no per-image API for individuals. Commercial use allowed at all paid tiers; free tier discontinued.

## Models

- **V6** (Dec 2023): first version with credible photoreal output and legible text rendering.
- **V6.1** (Jul 2024): coherence and faces.
- **V7** (Apr 3 2025; default Jun 17 2025): rebuilt architecture, 20-30% faster than V6.1; introduced Draft Mode (cheap fast preview) and Omni Reference (character / object consistency from a reference image). Added voice prompting.
- **Video** (Jun 2025): image-to-video, 5-21 second clips. First-party video, no longer image-only.
- **V8 / V8.1** (Apr 30 2026 on web): fastest model so far, ~4-5x speedup on standard jobs.

## Distinctive trait

The aesthetic baseline. Default outputs are stylized, painterly, "Midjourney-looking" even when prompted for photoreal. Strongest on illustration, concept art, mood pieces. Weaker than gpt-image-1 / Imagen 4 on instruction-following and embedded text. Omni Reference closed the character-consistency gap with FLUX Kontext.

## Maturity

Production. Used by ad agencies, indie game devs, book illustrators. Largest single-vendor consumer base for AI imagery.

## Individual empowerment

$10/mo replaces a stock-image subscription (Shutterstock $30+/mo) and gives a solo creator unlimited concept art, blog header images, book covers. A hobbyist illustrator can prototype 50 character variants in an afternoon that would have taken a week of hand sketching.
