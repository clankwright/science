## What it is

Early entrant in the consumer browser-agent space (2023-2024). Originally a Chrome extension that let users issue natural-language commands to drive their own logged-in browser. Pivoted in 2024 toward an API-first product (Agent V1 Beta, Agent 2.0 with their AgentQ research) and rebranded the parent company to AGI Inc. ("the AGI company"). Built on a proprietary ACE (Autonomous Cognition Engine) combining vision, language, and web interaction.

## Vendor / Access

- Vendor: AGI Inc. (formerly MultiOn)
- Access: API key signup at multion.ai; Chrome extension still available
- Sessions run either in the user's browser (extension) or in MultiOn's cloud (API)

## Cost

- Free tier on the extension for limited tasks
- API: usage-based, pricing not publicly listed in standard tiers; enterprise-flavored sales motion
- Heavy users have generally migrated to Browser Use (open source) or ChatGPT Agent (built into existing subscription)

## Maturity

- Beta. Product velocity has slowed visibly compared to 2023-2024 as competitors (ChatGPT Agent, Browser Use, Mariner) matched or exceeded capability.
- AgentQ research paper (RL self-improvement for web agents) was a notable contribution; product traction less so.

## Distinctive trait

First-mover. The Chrome extension format was novel in 2023: "agent runs in *your* logged-in browser, not a cloud sandbox" sidesteps the entire authentication problem at the cost of trusting the model with your own session.

## Individual-empowerment angle

Niche in 2026. The original use case (logged-in personal admin via a browser extension) has been partially absorbed by ChatGPT Agent's connector model and by Browser Use's local Playwright sessions. MultiOn remains relevant mainly as a CrewAI / LangChain integration tool for developers who already standardized on it.

## Adjacent: agent.ai

Sometimes confused with MultiOn. agent.ai is HubSpot founder Dharmesh Shah's no-code agent marketplace (different product, different team, no relation to MultiOn / AGI Inc.). Out of scope for this category page.
