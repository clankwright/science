# Robinhood Agentic Trading
Vendor: Robinhood | Date: 2026-05 | Maturity: beta
URL: https://news.bitcoin.com/robinhood-launches-ai-agent-trading-for-27-million-customers-options-and-crypto-next/

- Lets an external AI agent (Claude or ChatGPT) place real equity trades inside a ring-fenced Robinhood account via Robinhood's Model Context Protocol (MCP) servers.
- Live P/L feeds, per-trade push notifications, one-tap disconnect.
- US only; funded Robinhood customer required; equities only at launch (options/crypto/futures planned). Robinhood Gold $5/mo optional. No added fee.
- First mass-market retail broker to expose live trade execution to consumer LLM agents over MCP. Reaches 27M customers.
- Caveat: does not change the Barber-Odean structural reality (retail mostly loses); this democratizes execution, not alpha.
