# Bolt.new (StackBlitz)

**Date verified:** 2026-05-03

## Vendor

StackBlitz (San Francisco / global remote). Founder & CEO Eric Simons. The company has been building browser-based dev environments since 2017; WebContainers (Node-in-the-browser via WASM) is the underlying technology.

## What it does

User types a description ("a Notion-clone with auth and dark mode"); Bolt generates a full Next.js / Vite / Astro / etc. project, runs it in a browser tab via WebContainers, lets the user iterate in chat, and deploys with one click. Bolt V2 added Bolt Cloud: bundled databases, auth, file storage, edge functions, analytics, hosting.

Differentiator: the entire stack runs in the browser. No remote VM, no install, no `npm install` waits. The full source is visible and editable; you can drop into the file tree at any time.

## Access

- Web: https://bolt.new
- Open source: https://github.com/stackblitz/bolt.new (the OSS version is the prompt UI; WebContainers is proprietary)
- Sign-in required; no waitlist as of May 2026.

## Cost

- Free tier: daily + monthly token allowance, no credit card.
- Pro: $20/mo (per pricing-page reports; some sources list $25/mo).
- Token-metered above the included quota; unused tokens roll over one month (since July 2025).

Cost tier: **low-to-mid** for individual use; can hit **mid-to-high** for active prototyping ($50-150/mo reported).

## Maturity

**Beta-to-production for prototypes.** Production-grade for landing pages, dashboards, CRUD tools, internal tools, hackathon projects. Reviewers consistently flag that it pairs well with a "real" IDE for the maintenance phase.

## Distinctive trait

Browser-native execution + visible full source. Sits between "I never want to see code" (Lovable, Base44) and "I want AI in my IDE" (Cursor). You can use Bolt without writing a line, then export to GitHub and continue in Cursor.

## Target user

Solo founders, hackathon participants, designers prototyping with real data, developers skipping setup for greenfield projects.

## Growth

Day-one ARR: $60k. Month one: $4M ARR. Month three: $20M ARR. Month five (Feb 2025): $40M ARR. One of the fastest-growing software products ever measured.
