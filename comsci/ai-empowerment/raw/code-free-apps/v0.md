# v0 (Vercel)

**Date verified:** 2026-05-03

## Vendor

Vercel. Launched late 2023 as a UI-component generator (shadcn/ui + Tailwind + React); evolved through 2024-2026 into a full-stack app builder with Vercel deploys.

## What it does

Original: prompt to generate React + Tailwind + shadcn UI snippets. As of 2026: full Next.js apps with sandbox runtime, Git panel for branch creation and PRs from chat, database integrations (Snowflake, AWS Postgres, etc.), token-billing model, Figma import. Output is production-ready Next.js code that runs natively on Vercel.

## Access

- Web: https://v0.app
- Account required; no waitlist.
- API: available on Premium and above.

## Cost (2026)

| Plan | Price | Credits |
|---|---|---|
| Free | $0 | $5/mo credit |
| Premium | $20/mo | $20/mo credit |
| Team | $30/user/mo | shared credits |
| Business | $100/user/mo | data not used for training |
| Enterprise | custom | SSO, priority |

Token-metered (input + output). Credits do not roll over.

Cost tier: **low** for solo, **mid** for teams.

## Maturity

**Production for UI generation; beta-to-production for full apps.** The UI-generation use case (designer or PM produces a clickable component) is rock-solid. The full-app generation is newer; works well for Next.js + shadcn stacks (the path Vercel has optimized) and weaker for anything else.

## Distinctive trait

Stack alignment. v0 generates the exact code that Vercel runs best: Next.js, shadcn/ui, Tailwind. No translation layer between "generated app" and "deployed app." Output is genuinely the same code you'd hand-write.

## Target user

Frontend developers (massive overlap with the existing Vercel/Next.js userbase), designers who want production code instead of Figma handoff, PMs prototyping flows.
