# Newer Entrants: Tempo Labs, Softgen, Dyad, Base44

**Date verified:** 2026-05-03

The category is crowded; below is the second tier of tools that matter for the wiki, each with a distinct angle.

## Base44

**Vendor:** founded December 2024 by Maor Shlomo (solo, Tel Aviv). Acquired by Wix June 2025 for **$80M cash** + earnouts through 2029. Six months old at acquisition, 250k users, ~$200k/mo profit, zero employees other than the founder, no investors.

**Product:** prompt-to-app for full-stack web + native mobile (iOS / Android publishing built in). Bundles DB, auth, hosting; no external service required. Best for fast MVPs and internal tools per reviewer testing.

**Why it matters:** Base44 is the canonical proof point for the solo-founder-vibe-coder thesis. The acquisition was widely cited as the moment Wix admitted vibe coding had eaten its category. Founder publicly said "I built software that builds software."

**Cost:** free tier; paid plans within typical $20-100/mo band for the category.

## Tempo Labs

**Vendor:** Y Combinator, San Francisco. Co-founders include former Figma/Stripe folk.

**Product:** "Visual Editor for React, powered by AI." Drag-and-drop editor that produces real React code; aimed at PMs, designers, and engineers collaborating on the same artifact. Generates 60-80% of frontend code via AI; integrates with VSCode, GitHub, Figma. Supports Expo for React Native mobile.

**Why it matters:** the design-tool-but-actually-code angle. Most other tools in the category are chat-driven; Tempo is mouse-driven on a canvas, which lands differently with designers. Pricing skews toward business: free tier (30 prompts), then a sharp jump to **Agent+ at $4,000/mo** with weekly delivery guarantees backed by human engineers.

**Maturity:** beta. The visual editing is real; the AI generation works; the human-backed Agent+ tier is unusual and signals Tempo isn't fully confident in pure-AI output yet.

## Softgen.ai

**Vendor:** small team / solo (limited public info).

**Product:** conversational full-stack web app builder. DB, auth, APIs included.

**Why it matters:** **pricing**. $33/year subscription + pay-as-you-go AI tokens at "30-50% cheaper than other AI tools" (wholesale-rate token billing). The cheapest serious entrant in the category by an order of magnitude on the subscription side. Designed for budget-conscious solo builders.

**Maturity:** beta. Reviews describe it as solid for MVP work; not a Lovable/Bolt-grade reputation yet but punches above its weight on cost.

## Dyad

**Vendor:** open-source project.

**Product:** full-stack AI app builder that runs **entirely locally** on the user's machine. Uses local LLMs (Ollama, LM Studio) or the user's own API key for cloud models. Generates apps similarly to Bolt / Lovable but with no cloud dependency and no data leaving the user's computer.

**Why it matters:** the privacy / sovereignty angle. The only credible offline option in this category. Best for developers and privacy-conscious builders willing to handle setup and deployment manually.

**Cost:** **free** (open source); user provides compute and any model API costs.

**Maturity:** preview-to-beta. Setup friction is real; not for the non-technical user.
