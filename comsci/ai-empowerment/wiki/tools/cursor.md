# Cursor

> **Summary:** AI-first code editor (VS Code fork) with agent mode, Composer 2 in-house frontier model (March 2026), and Agents Window for cloud-parallel runs. The leading IDE-native agentic coding product. Composer 2 release made power-user workflows viable on Pro for ~$20/mo.

**Sources:** [[raw/agentic-coding/cursor.md]], [[raw/agentic-coding/00-overview.md]]

## What it does

Inline AI completion, chat sidebar with codebase context, agent mode for multi-step tasks, Composer 2 for long-horizon coding rollouts trained on hundreds of actions per task. Agents Window dispatches parallel cloud runs, each on its own worktree.

## Access and cost

Pro $20/mo (most users); Business and Enterprise tiers above. Free tier available with limited agent runs. Bring-your-own-API-key option for power users.

## Distinctive trait

Best-in-class IDE UX in the category; the only major agentic-coding product with its own frontier model (Composer 2). Cloud Agents Window unlocks parallel work that single-context CLIs cannot match.

## Limits

- IDE-only (no CLI surface as of May 2026).
- Composer 2 is excellent on Pro but power users hit usage caps faster than with Claude Code.
- Less open than Cline / Continue / Roo Code; harder to swap models freely.
