# Bolt.new

> **Summary:** Browser-native code-free app builder running on StackBlitz's WebContainers (Node in the browser tab). Best developer-adjacent experience in the vibe-coding category: users see and own the code, no setup friction. Hit $4M ARR in 4 weeks, $40M in 5 months.

**Sources:** [[raw/code-free-apps/bolt-new.md]], [[raw/code-free-apps/00-overview.md]]

## What it does

In-browser dev environment with AI-generated full-stack apps. Live preview, integrated package management, instant deploy. Users can drop into the code at any time and edit directly; pairs well with [Cursor](cursor.md) for the maintenance phase after the prompt-driven build.

## Access and cost

Free tier with limited tokens. Pro $20/mo. Pro+ $50/mo. Deploy through Bolt Cloud or export.

## Distinctive trait

WebContainers eliminate setup entirely; the dev environment runs in the user's browser tab. Lowest friction of any code-free tool for users who want code-visibility without leaving the build flow.

## Limits

- Same security profile as other vibe-coding tools (see [code-free apps capability page](../capabilities/code-free-app-building.md)).
- WebContainers limit some native dependencies (binary npm packages).
- Token usage adds up; heavy users hit ceilings.
